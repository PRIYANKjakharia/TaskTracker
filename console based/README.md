# Task Tracker Application
## Technology Used
Core Java ( console based application )
## Features Completed
- Add task
- View task list
- Mark task as completed
- Delete task
## How to Run
1. open the project in a java ide
2. compile java file
3. run main class
4. use console menu to navigate
- use the javac command to copile and java command to execute
eg: javac Main.java
    java Main```
## Challenges Faced
- designing the application structure by seperating task data and task management logic into different classes
- choosing and appropriate data structure for storing tasks and implementing efficient retrieval and deletion
- managing task operations using unique task IDs while keeping it minimal for easy deletion entry
